TyFast —— 后端开发者的 Vue 3 快速开发框架

一套基于 Vue 3 + Vuetify 的后台管理系统快速开发框架，专为后端开发者设计。 它的核心理念是：用 Vue，但不需要学 Node.js、Webpack、Vite——你只需要会 Java Web 和 Thymeleaf，就能直接上手 Vue 3 组件开发。

一套代码同时适配 PC 端、移动端和小程序端，并支持多租户模式。

定位： 让后端开发者零前端工程化负担，快速搭建管理系统和数据大屏。

TyCode —— 模板驱动的通用代码生成器

一款“Write once, Generate more”的代码生成工具。 只要你会写 Thymeleaf 模板，TyCode 就能帮你把重复的 CRUD 工作量降到零——支持 Java、Vue、SQL、Python、Go 等任意语言。

支持 MySQL、Oracle、SQLServer 等多种数据库，模板和数据源支持热加载（修改无需重启），并已支持 GraalVM Native-Image 原生编译，可编译为独立二进制文件，毫秒级启动。

定位： 让重复工作归零，统一团队代码规范，规避 AI 生成代码的不可控性。

TyEdu —— K12 精准教学系统

TyEdu 是一款面向 K12 教育阶段的开源免费精准教学系统，致力于解决大班教学中“因材施教”难以落地的现实困境。 系统通过数据采集与智能分析，为每位学生绘制学情画像，帮助实现“精准学、精准补、精准练”。

核心功能包括：题库管理（智能组卷）、考试分析与报告、错题集（含变式推荐）、学情分析（雷达图、进步曲线、偏科预警）、学习技巧库等。基于 TyFast 开发，使用 SQLite 数据库和 Apache Lucene 作为推荐引擎核心。

定位： 让数据照亮教育，帮助孩子少走弯路。

CVMonitor —— 基于 BoofCV 的视频变化检测

CVMonitor 是一个基于开源计算机视觉库 BoofCV 实现的摄像头视频内容变化检测工具。 通过 FFmpeg 拉取摄像头 RTSP 主码流，每秒取一帧图片，基于 BoofCV 判断画面是否发生变化。

变化的帧图片经 Base64 编码后通过 REST API 发送给 AI 服务，可进一步做人员入侵检测、人脸检测与识别等处理。算法采用 MSE 与 NMSE 评价画面变化，可输出图片相似度。

定位： 为 AI 视觉应用提供高效、轻量的视频帧变化检测前置能力。

TyOfficeTools —— 开箱即用的报表导出工具

TyOfficeTools 是一套高性能、开箱即用的报表导出工具，全面覆盖 Word、Excel、PDF 等主流报表格式。

核心设计理念是 “所见即所得” ：使用熟悉的 Office 工具在 docx 文件中设计版面，通过模板渲染数据，大幅降低 Hard Coding。Word 报表基于 poi-tl，Excel 报表基于 JXLS，PDF 通过 aspose.words 将 Word 转换为 PDF。

定位： 统一报表导出方案，降低多语言系统维护成本。

Paddle-TS-Serving —— PaddlePaddle 时序模型服务化部署套件

Paddle-TS-Serving 是 PaddlePaddle 时序模型的高性能端到端在线推理服务，支持 RESTful 协议。 与官方 Paddle Serving 不同的是，新增了对 TS 模型协变量的支持。

提供 Docker 镜像快速体验，免去复杂环境配置。RESTful API 包括健康检查接口（/health）和推理接口（/time-series-forecasting），支持 CSV 格式的时序预测输入输出。

定位： 让 PaddlePaddle 时序模型的部署和服务化变得简单高效。
